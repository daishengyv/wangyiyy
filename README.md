# wangyiyy
使用react 网易云的api 的音乐播放器


环境要求

需要 NodeJS 6.0+ 环境

安装

`$ git clone git@github.com:daishengyv/wangyiyy.git`
`$ npm install`


运行

$ npm run dev
服务器启动默认端口为8080

$ npm start
服务器启动默认端口为3000,请在3000 端口浏览
